Binden Sie den VeasySport Shop einfach in Ihre eigene Webseite ein.

== Description ==

Mit diesem Plugin kann der VeasySport Shop einfach in die eigene Webseite eingebunden werden.

= Funktionen =

VeasySport für Wordpress enthält folgende Funktionen

* Anzeige der Shop-Seite in Beiträgen oder Seiten mithilfe des [Gutenberg Embed-Blocks](https://wordpress.org/documentation/article/embed-block/).

= Installation =

1. Rufen Sie in WordPress die Funktion **Plugins > Neu hinzufügen** auf.
2. Suchen Sie nach **VeasySport**.
3. Installieren und aktivieren Sie das VeasySport Plugin.
4. Erstellen Sie einen Beitrag oder eine Seite und fügen Sie dort die Einbettungs-URL für
   Ihren VeasySport Shop ein. Das Plugin erkennt automatisch, dass Sie den Shop einbinden möchten und lädt den
   Shop mithilfe des [Gutenberg Embed-Blocks](https://wordpress.org/documentation/article/embed-block/).
